概要

オンラインハッカソン『Boost』にて制作しました。
主にハッカソンやzoomの背景、自己紹介として使える画像が簡単に作成できる技術者向けwebサービスです。
4人チームで自分はデザインとフロント開発の一部を担当しました。
ハッカソン中には完成できなかったため、現在自分が個人開発で完成を目指しています。

制作期間

2020年3月12日〜14日(2日半)

使用技術

HTML / CSS / Sass / Javascript / Vue.js / Figma / (Ruby on Rails)

制作背景・コンセプト

今回のハッカソンのテーマが「Boost」でした。
同じチームになったメンバー達とmiroを用いて、「Boost」の解釈と制作の方向性を決めるアイデアソンから行いました。

ハッカソンは短期間であるため、チームのメンバーといかに早く打ち解け、互いの技術スタックを確認できるかが大事である。

という結論のもと、「Boost」する対象をオンラインハッカソン参加者として、コミュニケーションを加速する(=Boostする)ツールの制作という方向性に決まりました。
また、アイデアソンの段階では既存の名刺背景の画像生成サービスを参考に、よりハッカソンに特化させたものとして、
discord経由でプロフィールの作成、共有、チームで共通の画像の作成によって団結感を高める(=Boostする)といったゴールができました。

工夫点

ハッカソンに特化した自己紹介カード作成ツールということで、得意領域や言語などの技術スタックを載せることができます。

現在、Vue.jsの勉強をしながら双方向データバインディングによる表示の実現に向けて開発中です。
