Balloon Mail

自分だけの発信の場になるSNSwebアプリです。

実際のサイトはこちらから

概要

大学の学生主体の作品展示会「卓展」2020で制作しました。
初めてのwebアプリ制作で、初めてRuby on Railsに触れました。
遠くの知らない誰かに手紙を送るというボトルメールに着想を得た、現代人に身近であるSNSを媒体としたwebアプリです。

制作期間

2020年5月〜8月

使用技術

HTML / CSS / Sass / Javascript / Ruby on Rails / illustrator / heroku

制作背景・コンセプト

卓展に向けて作品を制作するにあたり、webアプリを制作したいという思いがありました。
テーマは制作当時、ニュースなどで取り上げられていた「SNSによる誹謗中傷」にスポットを当てました。

匿名性を利用した批判。数値、視覚化された承認欲求。
それらにとらわれない自分だけの場所で、身近な人には直接言えないけど聞いてほしい。溜め込んだことを吐き出して発散したい。誰かに自分のことを伝えたい。
そういった発信の場になるSNSがこのアプリのコンセプトです。

できること

このアプリでできることは、投稿をする、人の投稿を見る、投稿をブックマークするだけです。
投稿へのコメントやユーザー同士の繋がりはありません。
知らない誰かの手紙に出会うボトルメールのように、表示される他の人の投稿はランダムです。
ひとりごとを投稿する。他の人のひとりごとをたまに探して見て、共感したらその手紙を保存しておく。
これだけのシンプルなものです。

デザイン・工夫点

アイデアの基になったボトルメールの、手紙の入った瓶が海を漂流するイメージの電子版として、
誰かの投稿がネットの海に漂うことを表現して空の水色と、ポジティブと暖かみを感じるオレンジを基調としました。

レスポンシブ対応しており、手紙のサイズを変えても文字が罫線に沿って表示されるようにしました。
